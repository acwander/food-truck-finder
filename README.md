## Steps to use this program

- Make sure you have node installed.
  - To check if you have Node.js installed, run this command in your terminal: `node -v`
  - If do not have node installed, install it with `brew install node`. Assuming homebrew is installed on your machine.
- Make sure you have npm installed.
  - To confirm that you have npm installed you can run this command in your terminal: `npm -v`
  - To update your npm, type this into your terminal: `npm install npm@latest -g`
- To run this program follow these steps.
  - In terminal, navigate to the proper directory the files are stored.
  - Install dependencies by typing this into your terminal: `npm install request`
  - Run the program by typing this into your terminal: `node FoodTruckFinder.js`
